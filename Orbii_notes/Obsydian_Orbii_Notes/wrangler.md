# What is *wrangler*?
"The remote control to *run and deploy* my *Cloudflare code*"

*Wrangler* is **Cloudflare's command-line tool** (*CLI*).
1. ***wrangler dev***: Run my Cloudflare WOrker locally for testing
2. ***wrangler deploy***: Send my code to Cloudflare so it runs on the internet
